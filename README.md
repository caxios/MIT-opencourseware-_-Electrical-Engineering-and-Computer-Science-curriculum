# MIT-opencourseware-_-Electrical-Engineering-and-Computer-Science-curriculum
MIT opencourseware _ Electrical Engineering and Computer Science curriculum for non-CS background people. 

I, as a non-CS backgroun person, selected some of the courses that are available from MIT OpenCourseWare to teach myself. 

I try to chose courses that there is enough materials to to study/practices, so I can enhance my CS skill/knowledge.

Further courses can be found from MIT opencourseware website : https://ocw.mit.edu

Following courses I have chosen are not listed sequentially, which means the order of the list is not neccessarily the recommended or actual curriculam, rather I just randomly lined up courses.
